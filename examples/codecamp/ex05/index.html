Let's create a Spaceship!

So in this lesson, we are going to apply what we have learnt in order to create a spaceship object.

Open up ex05/index.html and take a few minutes to study the code.

Let's take a look at 3 things:

  • How to create the spaceship object
  • Giving the spaceship some properties
  • Giving the spaceship some functionality

Create the Ship

        /* Create my player object */
        var spaceShip = {};

A quick recap tells us that we just created a variable called 'spaceShip' and the variable is of type....what?
What kind of variable is it? Or another way of phrasing that question is: What are we storing in the variable so that we can find (access) it later?

What can we store in a variable? Take a look below:

var myNum = 10;/* A number */

var myStringValue ="abcdefg";/* A string of characters */

var myObject = {};/* An object */

var myVariableWithNothingDefinedAtAll;/* An undefined value */

Add Properties to the Ship

        /* Give my ship some properties */
        spaceShip.direction = 'right';
        spaceShip.speed = 5; // Where 1 = slow and 10 = fast
        spaceShip.image = 'ship.png';

Remember that properties are the same things as characteristics. Until we give our spaceship some characteristics, it is really nothing of any consequence at all. In the above code, we have given it direction, speed and an image. Well we have to start somewhere!

Add Methods to the Ship

        /*Shoot a bullet*/
        spaceShip.shoot = function(){
          console.log('Shoot a bullet...!');
        };

        /* Move left */
        spaceShip.moveLeft = function(){
          console.log('Moving left...!');
        }

        /* Move right */
        spaceShip.moveRight = function(){
          console.log('Moving right...!');
        }

This is where all the good stuff happens. Any object that you create must be able to do stuff. In the above code, we have given our ship the ability to shoot and move left or right. Obviously the methods themselves should be coded properly rather than just spit out a message to the console but again, it is a start!

Use the Ship

So how do we use the ship to shoot? Well, we have to call the 'shoot' method. Remember the 'dot' notation? Can anyone explain how to call the 'shoot' method in the code?

So, just to recap then:
  1. We looked at how we could create a spaceship object
  2. We created the spaceship, then gave it properties and methods
