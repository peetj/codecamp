Parameters are pieces of information that are passed into a function. In javascript you can pass as few or as many parameters as you like. The information that is passed could be anything, eg. strings, numbers, objects...etc. Let's look at a few examples: 


        /*

          easy to do! If you want you can have a go!
        */

        /* Call the function */
        add(2, 2);

        /*
          This function is probably part of a virtual telephone because it actually calls
          somebody through Skype. In order to do that, it needs a telephone number which is
          what is passed into the function.
        */
        function callPersonOnSkype(telNumber){
          /* Make the call */
        }

        /* Call the function */
        callPersonOnSkype("0428 334727");

In the first example, we are simply providing an 'add' function, much like that you would find on a calculator, the only difference being that you may want to add up more than 2 numbers so in that case you would have many parameters. Javascript allows us to do that easily.

In the second example, we have a function that invokes a Skype call. All the function needs is the person's telephone number. It can then do whatever it needs to do to call that person.
