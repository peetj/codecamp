Description

Congratulations, you made it to Part 2! In the first part if you recall we simply printed out the word 'Hello' to the screen using the console object. In this part we are going to introduce a variable.

Computer programs need to store and remember values so that they can use them again at a later time. They store these values in variables. In javascript we create a variable with the 'var' keyword. Here is an example:


        var dob = "08-02-1970";
The above example creates a variable called 'dob' and sets this to the value "08-02-1970" which happens to be my date of birth! Now as an example, the program may want to print out my age at some later time. It will use the 'dob' variable that it stored previously to do this.

Hopefully, you have a basic understanding of what a variable is at this point. A variable can store numbers, strings (a string is simply a sequence of characters in quotes like 'dob' above), objects and can have a couple of special values as well but we needn't worry about any of this at the moment. In our example below we are going to define a variable called 'name' and give it the value "Bob". Take a look at the code below. 


        function sayHello() {
          var name = "Bob";

          /* This will print hallo to the screen */
          //console.log( 'Hallo ' + name );
        }

        /* This next line will run the 'sayHallo' function */
        sayHello();

Open the file ex02/main.js in Sublime. At the same time open up the Chrome Developer Tools so that you can view this page and the console at the same time. Do you see any output on the console? Does it say 'Hello Bob'? Can you work out why there is no output shown? If not, go back to the javascript code in main.js in your Sublime text editor and check to see if anything is commented out that shouldn't be.

Fix the code and run the example again. You should see the text 'Hello Bob' on the console.

So, just to recap then:

  1. We looked at variables, what they are and why we need them
  2. We updated our 'sayHello' function to say hello to Bob
  3. We refreshed the page to incorporate the changes that we made
  4. We opened up the Chrome Developer Tools window to see the result of our program

In part 3 we will be saying goodbye to our sayHello() function by passing in our own value for the variable that we used in part 2. This means we will be able to say 'Hello' to whomever we want. This is surely a really useful program!
