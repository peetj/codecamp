How to Install Chrome

  1. Go to the following URL: Download Chrome
  2. Click on the 'Download Chrome' button
  3. Click on the 'Accept and Install' button.
  4. If you are on a Mac, the downloaded file will be called 'googlechrome.dmg'. Simply double-click on the file and copy the Chrome application into your 'Applications' directory.
  5. If you are on Windows the downloaded file will be called 'ChromeSetup.exe'. Simply double-click on the file and follow the instructions
