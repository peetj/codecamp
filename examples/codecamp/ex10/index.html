A spaceship is no good unless it can fire bullets - right? Well that's what this lesson is all about. Check out the example in game.html in the ex10 folder.

So this time, we are going to list out the steps that it takes to add a shooting capability to our ship. You can then go through the code and ponder over which bit does what!

Here are the things we need to do to get bullets !!!

  • Bind the ship entity to the 'KEY_DOWN' event
  • If the user presses the SPACE KEY, trigger the shooting of a bullet
  • Before creating the bullet, set the (x, y) position of where it will be
  • Create a bullet entity using Crafty.e()
  • The bullet will be a simple rectangle with position, width, height and color
  • Animate the bullet using the Tween component

Head straight to the code in main.js in the ex10 folder and try and work out which line is doing what. Please make sure you ask questions if you get stuck.

When you have a reasonable idea of how the bullet code works, try and modify it (see below).

So, just to recap then:
  1. We learned how to give our ship the ability to shoot bullets

If you have time try to modify the bullet behaviour:
* The initial position of the bullet
* The bullet color and size
* How fast the bullet travels
