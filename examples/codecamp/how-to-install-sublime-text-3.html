How to Install Sublime Text 3

  1. Go to the following URL: Download Sublime Text 3
  2. Click on the appropriate link for your operating system (OS X, Windows or Windows 64 bit) button. If you don't know whether you have a 64-bit Windows operating system then click on this link from Microsoft which will help you figure it out
  3. Once the file is downloaded, click on it to run it
  4. Follow the installation instructions - NB. If you get an option to 'Add to explorer context menu' then take it!
  5. Goto the Start menu if you are on windows or the Applications folder if you are on Mac and start up Sublime
  6. You then need to do a couple of things
    • Goto Preferences -> Color Scheme and choose LAZY
    • Goto Project -> Add Folder to Project... and choose the 'codecamp' folder that we previously copied to the Desktop
  7. Don't worry, if you exit Sublime and relaunch it, your settings will be remembered!!
