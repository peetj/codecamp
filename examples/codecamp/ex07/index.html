This will contain the game that we are developing at each stage of development. So you will need to open a new tab in Chrome (CTRL+T on Windows, âŒ˜+T on Mac) and open the game.html file. The javascript for the game will be in main.js as before.

Check out the code below. This will initialise the Game engine and our game stage. 


        /* Initialise variables */
        var _screenWidth = 800;
        var _screenHeight = 600;

        /* Initialise Game Engine */
        Crafty.init(_screenWidth, _screenHeight, document.getElementById('gamecanvas'));

        /* Load Assets */
        Crafty.background('rgb(0,0,0)');

Check out the 2 variables for width & height. This sets the screen dimensions.

Next we initialise the Crafty game engine by calling a function init(). This stands for 'initialise'. Notice that we passed the width, height and something else. The something else is the HTML DIV element which has an id 'gamecanvas'. Flick to the game.html file in Sublime and you will see the element being used. Without game initialisation, we can do nothing.

Lastly, we set the background color using 'rgb' values. rgb means red, green & blue. We give each one a number between 0 and 255 to 'make' a color. In our case we have used (0, 0, 0) which is black.

Use the next few minutes to play around with the dimensions of the game stage and the color of the background. Every time you make a change and save it, you will need to flick to the browser tab for game.html and refresh it to see the result of your changes.

So, just to recap then:
  1. The first thing we need to do is initialise the CraftyJS game engine
  2. To set the background we can give it a color based on the RGB system
  3. RGB simple adds a color with some red in it, some green in it and some blue in it
