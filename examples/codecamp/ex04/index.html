Its all about Objects - just think about it. We spend the whole day, every day, 7 days a week interacting with/playing with/fiddling with....objects!!

What is an object?

It is critical that we grasp the concept of an object if we are going to program anything of any consequence.

In this lesson we are going to build a calculator. It sounds complex but actually it isn't as difficult as it sounds. Firstly we are going to discover how to create an object in javascript. Note that javascript is a very versatile language and there are several ways to create an object. Whilst we may describe the different ways to create an object, we will the concentrate on a particular one.

Look at the code fragment below: 


        /* Example 1 creates an object using the object literal notation {} */
        var object-1 = {};

        /* You can also create an object with the Object() function */
        var object-2 = new Object();

        /* Finally we see that a function in Javascript is actually an object as well */
        var object-3 = function(){ // does nothing };

So object-3 doesn't really do a fat lot yet but we can give it some power! Not much I grant you but we can at least make it a bit more interesting.

Let's change its name to 'Fido' - if this sounds a bit strange just bear with me! 


        var Fido = {};

Ok - that's not too hard is it? We have created a variable called 'Fido'. The variable happens to be an object so it is in fact an object variable (as opposed to a numeric variable or a string variable).

Now, Fido can't really do anything and doesn't really have any characteristics. Do you have a pet? What kind of characteristics does your pet have eg. colour, weight, hairyness...etc! Let's give Fido some characteristics. How about colour? 


        var Fido = {};

        Fido.colour = "brown";

Note the 'dot' notation. After the name of the object (Fido) we place a dot (Fido.) The property name "brown" comes after this. "brown" is a string value. Any text value is a string value and must be placed in quotes thus "". You might be thinking that Fido is a dog. I wonder why? Anyway, lets give Fido a property called "breed" 


        var Fido = {};

        Fido.colour = "brown";

        Fido.breed = "spoodle";

When we talk about properties of objects, this is the same as saying that the object has certain characteristics. In other words, a property is a characteristic.

Finally, we are going to add a method to Fido because the poor thing can't actually do anything yet. We are going to add a method/function called 'talk'. 


        var Fido = {};

        Fido.colour = "brown";

        Fido.breed = "spoodle";

        Fido.talk = function(){ console.log("Bark! Bark!"); };

So, we have a fully functioning dog! Ok maybe not fully functioning but you get the picture.
